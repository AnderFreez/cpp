1) принимает натуральное число, возвращает сумму нечётных цифр.
2) принимает натуральное число n, возвращает сумму чисел от 1 до n, то есть 1 + 2 + 3 + 4 + .... + n.
   Например, принимает 5, возвращает 1 + 2 + 3 + 4 + 5.
3) принимает натуральное число n, возвращает сумму чётных чисел из диапазона от 1 до n, то есть 2 + 4 + 6 + 8 + .... .
   Например, принимает 7, возвращает 2 + 4 + 6.
4) принимает натуральное число, возвращает сумму цифр, но со следющими знаками:
   например, 123 принимает, 3 - 2 + 1 возвращает, 1234 принимает, 4 - 3 + 2 - 1 возвращает.
   То есть цифра младшего разряда берётся с плюсом, а далее знаки чередуются.
5) принимает натуральное n, возвращает 1! + 2! + 3! + ... + n!.
   Например, примает 5, возвращает 1! + 2! + 3! + 4! + 5!.
   Напоминание: 4! = 1 * 2 * 3 * 4.
   Указание: эту функцию очень легко написать, используя factorial, который уже написан.
   Также будет полезно использовать идею суммирования из задач 2, 3.
